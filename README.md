# Learn React

This repository contains all the code, exercises, and mini-projects I completed while following a full React.js course.

The purpose of this repository is to document my learning journey and serve as a reference for the React concepts I studied throughout the course.

## Topics Covered

* React fundamentals
* Project setup with Vite
* JSX
* Functional components
* Props
* State (`useState`)
* Event handling
* Conditional rendering
* Rendering lists
* Forms
* React Hooks
* React Router
* Context API
* API fetching
* Component composition
* React best practices

## What This Repository Contains

* Course exercises
* Practice examples
* Mini projects
* Code snippets used to understand React concepts

Each folder or project represents a different lesson or concept covered during the course.

## Outcome

By completing this course, I gained a solid understanding of React fundamentals and built a strong foundation for creating modern React applications.

This repository marks the completion of my React learning course. Future React projects and applications will be developed in separate repositories as I continue to improve my skills.
